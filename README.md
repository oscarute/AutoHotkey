# Apps for Windows

## OScript

This script allows to reboot and shutdown using hotkeys in computers where the interface is locked (Kiosk mode).

### Automatic execution
Save the executable file and create a shortcut in your startup folder (C:\Users\<UserName>\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup).

### Usage
Shutdown: Control + R 
Reboot: Control + S  
Cancel: Control + A

Tip: Once the user has enabled shutdown or reboot has 30 seconds to cancel.
